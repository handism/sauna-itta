# 機能追加案: 地点検索機能の導入

## 概要
現在は地図上を直接クリックして場所を選択する必要がありますが、施設名や住所で検索してピンを立てられるようにします。

## 課題
- 知らない土地のサウナを登録する際、地図上の正確な位置を特定するのが難しい。
- 登録の手間を減らしたい。

## 解決策
- Nominatim (OpenStreetMap) や Google Maps Geocoding API 等を利用した検索窓を実装する。
- 検索結果を選択すると、その地点に地図が移動し、ピンが仮設置されるようにする。

## 実装イメージ
- `LocationPicker` コンポーネントに検索バーを追加。
- 検索結果をリスト表示し、選択時に `onLocationSelect` を呼び出す。
