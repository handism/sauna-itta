# 機能追加案: PWA (Progressive Web App) 対応

## 概要
スマートフォンでネイティブアプリのようにインストール可能にし、オフラインでも地図が見られる（キャッシュされた範囲）ようにします。

## 課題
- サウナ施設内や地下など、電波の不安定な場所で記録を付けたい場合に不便。
- ホーム画面からワンタップで起動したい。

## 解決策
- `next-pwa` 等を使用して `manifest.json` と Service Worker を設定する。
- 静的アセットと API レスポンス（外部地図タイルは要検討）をキャッシュする。

## 実装イメージ
- `public/manifest.json` の作成。
- `next.config.js` への PWA 設定の追加。
