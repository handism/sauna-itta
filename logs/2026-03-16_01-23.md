### CIビルドエラーの修正

**ブランチ:** `issue-8-visit-calendar`

**エラー内容:**
`src/app/stats/page.tsx:202` の `Calendar` コンポーネントの `onChange` プロパティで TypeScript 型エラーが発生。

```
Type 'Dispatch<SetStateAction<Date | Date[]>>' is not assignable to type
'(value: Value, event: MouseEvent<HTMLButtonElement, MouseEvent>) => void'
```

`react-calendar` の `onChange` が受け取る `Value` 型は `null` を含むが、`setDate` の型 `Date | Date[]` が `null` を許容しないため型の不一致が発生。

**修正内容 (`src/app/stats/page.tsx`):**
1. `useState<Date | Date[]>` → `useState<Date | null>` に型を変更
2. `onChange={setDate}` → `onChange={(value) => setDate(value instanceof Date ? value : null)}` でラッパー関数に変更

**結果:** `npm run build` 成功
